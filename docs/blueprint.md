# **App Name**: Veiled Ventures

## Core Features:

- AI Anonymizer: AI-powered parsing and anonymization of job descriptions and candidate profiles. The tool will use an LLM to selectively reveal information to maintain anonymity while ensuring relevant details are communicated during each phase.
- Phased Reveal: Phased information reveal system. Gradually disclose candidate and employer details as engagement progresses, based on mutual interest and platform activity. Employers must provide a deposit when listing a job, with partial refunds tied to timely proof of hire.
- Tiered Messaging: Tiered messaging cost. Implement a pay-to-message system for candidates and employers, scaling costs with conversation duration and frequency to discourage bot activity and encourage commitment.
- AI Matching: Matching system that scores the match. Potential employee should take the Big 5, MBTI and location weighing heavily on the matching algo. Match jobs to personality type results that best match with the estimated ideal candidate MBTI, location, experience, profile tags, location all factoring in best match. LLM tools will be used. Also, the hiring manager for the company must take a MBTI and Big 5 quiz to ensure best compatibility with the potential employee for a positive work environment.

## Style Guidelines:

- Primary color: Neutral gray (#F5F5F5) to emphasize content.
- Secondary color: Light blue (#E0F7FA) for a professional feel.
- Accent color: Teal (#008080) for highlighting interactive elements and calls to action.
- Clean, card-based layout to present anonymized information in digestible chunks.
- Use a consistent set of minimalist icons to represent job functions and candidate skills.
- Subtle transitions and animations to indicate information reveals and messaging updates.
